# Green Argus Index: Project Instructions

## Project Overview

**Green Argus Index** is an AI-powered system for automated, objective assessment of educational video quality based on observable teaching indicators rather than brand, author charisma, or marketing promises.

### Core Scope
- **Input**: YouTube video(s) or video transcript + metadata
- **Output**: Green Argus rating profile (0–100, grade A–E, 8-scale assessment, red flags)
- **Methodology**: AI-driven feature extraction, segment analysis, risk detection

### Current State
- MVP exists with YouTube integration, 8-scale scoring, risk detection
- **Gap**: Implementation does not fully align with specification
  - Scale weights are incorrect (current vs. spec)
  - Red flag system needs hardening
  - Multi-video aggregation not implemented
  - Output format incomplete

## Superpowers Workflow

This project uses **Superpowers methodology**:

### Output Format (required)
1. Интегральный рейтинг 0–100 + класс A–E
2. Профиль по 8 шкалам (0–10 каждая)
3. Сильные стороны (2–4 пункта с таймкодами)
4. Риски и красные флаги
5. Для кого подходит
6. Дисклеймер о границах оценки

### Red Flags (must not average across videos)
- Завышенные обещания
- Подмена обучения мотивацией/продажей
- Псевдоэкспертность
- Непроверяемые ключевые утверждения
- Отсутствие ограничений
- Техническая непригодность
